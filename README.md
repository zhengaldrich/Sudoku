Sudoku Lab - Aldrich Zheng
This project can genreate a sudoku board and create a playable board. To run the game, simply run the code. If you just want to generate a solved sudoku board, you could make lines 8 and 10-30 into comments. 
The sudoku board is generated by a main fillBoard method, looping through a 1D array of numbers from 1-9 and filling each row with them.
The only file in this project is the main Sudoku.java file.
